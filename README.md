## 🚀 Getting Started

### Local Development

1. Clone or download the repository
2. Open `index.html` in your browser, or
3. Run a local server:
   ```bash
   python3 -m http.server 8000
   ```
4. Visit `http://localhost:8000`

### Customization

- **Colors**: Modify CSS custom properties in `:root` and `[data-theme="light"]`
- **Content**: Update text, links, and project details in the HTML
- **Contact Form**: Replace the Formspree endpoint with your own
- **GitHub Stats**: Update the username in the API calls
- **Resume**: Add your resume PDF and update the download link

## 📧 Contact Form Setup

The contact form uses Formspree for handling submissions. To set it up:

1. Sign up at [formspree.io](https://formspree.io)
2. Create a new form and get your endpoint URL
3. Replace the `action` attribute in the contact form with your endpoint
